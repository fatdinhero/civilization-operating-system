# Contributing to COS Backend

🕌 **Bismillah ir-Rahman ir-Rahim**

Thank you for your interest in contributing to the Civilization Operating System (COS) Backend!

## Code of Conduct

This project follows Islamic ethics and values:
- Respectful communication
- Honest collaboration
- Quality over speed
- Knowledge sharing as Sadaqah Jariyah

## Development Setup
```bash
# Clone the repository
git clone https://github.com/fatdinhero/civilization-operating-system.git
cd civilization-operating-system/backend

# Install dependencies
npm install

# Run tests
npm test

# Build
npm run build

# Lint
npm run lint

# Format code
npm run format
```

## Testing

- Write tests for all new features
- Maintain 70%+ code coverage
- Run `npm test` before committing

## Code Style

- TypeScript with strict mode
- ESLint + Prettier enforced
- Meaningful variable names
- JSDoc comments for public APIs

## Commit Messages

Use conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `test:` Tests
- `refactor:` Code refactoring
- `chore:` Maintenance

Example: `feat: add Murabaha contract validation`

## Pull Requests

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/amazing-feature`
3. Make your changes
4. Run tests: `npm test`
5. Commit: `git commit -m 'feat: add amazing feature'`
6. Push: `git push origin feature/amazing-feature`
7. Open a Pull Request
